# Image Files Directory

This directory contains all images for the GK Pvt Industries website.

## Required Images

Please add the following image files to this directory:

### 1. truck-1.jpg
- **Description**: Orange truck image with PB GK 115 registration
- **Use**: Displayed in Gallery section and Truck Rental service
- **Recommended size**: 1200x800px or larger
- **Format**: JPG, PNG, or WebP

### 2. steel-products.jpg
- **Description**: Steel nuts/products image (hexagonal metal nuts)
- **Use**: Displayed in Gallery section and Raw Steel/Iron Products services
- **Recommended size**: 1200x800px or larger
- **Format**: JPG, PNG, or WebP

### 3. workshop.jpg
- **Description**: Industrial workshop image showing manufacturing facility
- **Use**: Displayed in Gallery and Facilities sections
- **Recommended size**: 1200x800px or larger
- **Format**: JPG, PNG, or WebP

### 4. building.jpg
- **Description**: GK Industries building exterior image
- **Use**: Displayed in Gallery and Facilities sections, and Godown Services
- **Recommended size**: 1200x800px or larger
- **Format**: JPG, PNG, or WebP

## How to Add Images

1. Rename your image files to match the names above exactly (case-sensitive)
2. Copy the files to this `public/images/` directory
3. Ensure file names match exactly:
   - `truck-1.jpg` (with hyphen, not underscore)
   - `steel-products.jpg`
   - `workshop.jpg`
   - `building.jpg`
4. Refresh your website to see the images

## Image Optimization Tips

- Use compressed images for faster loading
- Recommended: JPEG for photos, PNG for graphics
- Maximum file size: 500KB per image (for web performance)
- Tools: Use tools like TinyPNG or ImageOptim to compress images

## Testing

After adding images, verify they appear in:
- Gallery section (homepage)
- Facilities section (building and workshop)
- Services section (relevant service cards)
